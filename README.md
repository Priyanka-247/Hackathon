## 🚀 AI Prompt Used for Building This Web Application

Create a **modern productivity dashboard web application** that helps users track their daily learning and self-improvement activities.

### 🎯 Objective

The webpage should function as a **technical productivity tracker** where users can monitor important daily tasks related to learning, coding practice, and personal development.

### 📋 Default Activities

Include the following core activities:

* Meditation
* Solving LeetCode Problems
* Solving Exercism Exercises
* Studying courses on Analytics Vidhya
* Completing AWS Courses
* Attending ByteXL Assessments

### 🎨 UI / UX Requirements

The design should be **professional and minimal**, avoiding childish colors or layouts.

Design features must include:

* Modern **dashboard style interface**
* Clean **card-based task layout**
* Soft **gradient background**
* Professional **color palette**
* Subtle **hover animations**
* Icons representing each activity
* Progress tracking system
* Fully responsive design

### 📊 Functional Features

The application must support:

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Daily productivity progress bar
* Task completion percentage
* Local Storage to save user progress

### 🧠 Additional Enhancements

Optional advanced features:

* Dark mode / Light mode toggle
* Motivational productivity messages
* Daily streak tracking
* Smooth UI animations
* Minimal and professional typography

### 🛠 Tech Stack

* HTML
* CSS
* JavaScript

### 💡 Expected Output

A visually attractive **productivity dashboard webpage** that helps users stay consistent with daily learning activities while maintaining a professional UI suitable for portfolio or GitHub showcase.
